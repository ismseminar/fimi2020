The Workshop on Functional Inference and Machine Intelligence (FIMI) is an international meeting for machine learning and nonparametric statistics. Especially, it focuses on the following topics:

  • Kernel Methods for Statistical and Numerical Analysis
  • Mathematical Analysis for Deep Learning with Neural Networks
  • Nonparametric Inference for Complex Data Structure

It will be held at EURECOM, Nice, during February 17th and 19th, 2020.

Previous Workshop: 2016, 2017, 2018, 2019
